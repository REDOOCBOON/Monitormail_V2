# How to Create a Google 16-digit App Password

If you are using Gmail to send email from MonitorMail, Google requires using an **App Password** when 2-Step Verification is enabled.

Follow these steps to create your 16-digit App Password:

1. **Sign in to your Google Account**
   - Visit: https://myaccount.google.com/

2. **Navigate to Security**
   - In the left menu, click **Security**.

3. **Enable 2-Step Verification (if not already enabled)**
   - Under **"Signing in to Google"** click **2-Step Verification** and follow the steps.

4. **Create an App Password**
   - After 2-Step Verification is enabled, return to **Security**.
   - Under **"Signing in to Google"**, click **App passwords**.

5. **Generate the App Password**
   - Under **"Select app"**, choose **Other (Custom name)**.
   - Enter a name like **MonitorMail** and click **Generate**.
   - Google will show a **16‑digit app password** in a yellow box.

6. **Copy and use the password**
   - Copy the 16-digit password (spaces are optional).
   - Use it as the **"Google App Password"** when sending email from MonitorMail.

---

### Notes
- This password is used **only** for MonitorMail (not your main Google password).
- If you ever need to revoke access, you can delete the App Password from the same page.
